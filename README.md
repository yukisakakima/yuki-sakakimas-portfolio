# Yuki Sakakima's Portfolio
## Description :
作品集を閲覧サイト。<br>
初代IBMパソコンが好きだったこともあり、そのようなデザインになっております。<br>
また、グラフィックには、初代IBMパソコンのコンセプトを崩さない為に、当時のスペックで可能なグラフィックとして幾何学模様を採用しております。
